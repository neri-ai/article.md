# テーブル設計書：記事情報（articles_info）

## 1. 概要

### 1.1. テーブル名  
* articles_info

### 1.2. 用途  
* 記事の基本情報（タイトル、投稿者、作成日など）を管理する。

### 1.3. 改版履歴  
| 日付 | バージョン | 対応者 | 対応内容 |
|--------|-------|------|-------|
| 2025//11/09 | 1.0 | 根來龍志 | 新規作成 |

---

## 2. テーブル構成
| 項目 | 内容 |
|-------|------|
| テーブル名 | articles_info |
| 主キー | article_id |
| 複合キー | user_id |
| キャラクタセット | utf8mb4 |

---

## 3. カラム使用

| カラム名 | データ型 | NULL 許可 | デフォルト値 | 備考 |
|-----------|----|------|------|------|
| article_id | VARCGAR(6) | No | - | 記事ID |
| user_id | VAECGAR(6) | No | - | 投稿者 |
| title | VARCHAR(255) | No | - | 記事タイトル |
| view_count | INT | N0 | 0 | 閲覧数（簡易カウント） |
| created_at | DATETIME | N0 | - | 作成日時 |
| updated_at | DATETIME | Yes | NULL | 更新日時 |
